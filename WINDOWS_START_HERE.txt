# Data Broker Removal Tool - Package for Windows

This folder contains everything you need to remove your data from data brokers on Windows.

## 🚀 QUICK START (Windows Users)

### First Time Setup:
1. **Install Python** if you don't have it:
   - Download: https://www.python.org/downloads/
   - ⚠️ CHECK "Add Python to PATH" during install!
   - Restart your computer

2. **Double-click:** `install_windows.bat`
   - This installs required libraries
   - Takes 1-2 minutes

### Using the Tool:

**Easy Mode (Recommended):**
- Double-click: `run_manual_windows.bat`
- Creates an interactive HTML checklist
- You manually click through each site
- Progress saved automatically

**Auto Mode (Semi-Automated):**
- Double-click: `run_windows.bat`
- Browser opens and helps fill forms
- You still handle CAPTCHAs
- Saves screenshots of everything

## ⚡ Fastest Way to Get Started

1. Right-click `install_windows.bat` → Run as administrator
2. Wait for it to finish
3. Double-click `run_manual_windows.bat`
4. Enter your info
5. Open the HTML file it creates
6. Start clicking through the list!

## 🆘 Troubleshooting

**"Python not found"**
→ Install Python from python.org (check "Add to PATH")

**"Permission denied"**
→ Right-click the .bat file → Run as administrator

**"Selenium error"**
→ Run `install_windows.bat` again
